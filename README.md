# たしざんかあど①  
## くりあがりのない たしざん

小学校低学年向けの「くりあがりのない足し算」を  
**視覚的・操作的に練習できる Web アプリ**です。

紙に書くことが負担になりやすい子どもや、  
デジタル教材の方が理解しやすい学習スタイルを想定して作られています。

---

## 🎯 対象

- 小学1年生〜
- くりあがりのない足し算の定着期
- 書字が負担になりやすい子
- タブレット・スマホでの学習が合う子

---

## ✨ 特徴

- 0〜10までの「くりあがりなし」の足し算のみ
- 視覚ヒント（●の数表示）ON / OFF 切り替え
- ランダム / 順番 / 逆順モード
- タイム・正答率・間違えた問題を自動記録
- ローカル保存（サーバー不要）

---

## 🧠 学習設計の考え方

- **書かせない**  
- **急がせない**
- **失敗を責めない**

間違えた回数ではなく  
「どの問題でつまずいたか」が分かる設計にしています。


---

## 🛠 技術メモ

- React（CDN）
- Tailwind CSS（CDN）
- Babel（ブラウザ変換）
- ビルド不要 / 単一HTML構成
